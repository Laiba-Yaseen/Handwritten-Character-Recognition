# ✍️ Handwritten Character Recognition using Deep Learning

## 📌 Project Overview

The **Handwritten Character Recognition** project is a Deep Learning-based application that identifies and classifies handwritten digits using a **Convolutional Neural Network (CNN)**. The model is trained on the **MNIST dataset** to accurately recognize handwritten digits from images.

---

## ✨ Key Features

* ✍️ Handwritten digit recognition
* 🖼️ Image preprocessing and normalization
* 🧠 CNN-based image classification
* 📊 Model training and evaluation
* 🔍 Prediction on unseen images

---

## 🛠️ Technologies Used

* Python
* TensorFlow/Keras
* NumPy
* Matplotlib
* Google Colab

---

## 🏗️ Project Workflow

1. Load the MNIST dataset
2. Preprocess and normalize images
3. Build and train the CNN model
4. Make predictions
5. Evaluate model performance

---

## 🚀 Applications

* Document processing
* Postal code recognition
* Bank cheque verification
* OCR systems
